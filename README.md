# ライブ配信エンゲージメント分析ツール

## プロジェクト概要
ライブ配信の7つのエンゲージメント指標データとテキスト分析から、美しい3セクション構成のレポートを自動生成するフロントエンドのみの静的Webアプリケーション。

## 主な機能

### ✅ 実装済み機能

#### データ入力
- **指標データファイル**（必須）: Excel (.xlsx/.xls) または CSV (.csv) をアップロード
  - 列名の揺れに強い自動マッピング（30+パターンの別名を吸収）
  - 対応指標: 経過時間、同時視聴ユーザー数、いいね数、シェア数、チャット数、商品クリック数、カート追加クリック数
- **コメントデータファイル**（任意）: Excel/CSV
- **「コメントデータをアップロードしない」オプション**（チェックボックス、デフォルト未チェック）
  - チェック時: コメント入力エリアを無効化し、指標データのみで分析を実行
  - コメントデータなし時は結果に通知バナーを表示
- **動画ファイル**（参照用・任意）: MP4/MOV/AVI（分析には使用しない）
- **テキスト分析データ**（テキストエリア）: マークダウン/プレーンテキスト対応

#### レポート構成（3大セクション）
1. **各指標の考察とアドバイス_1（単一指標分析）**
   - 分析ポイント①: 複合時系列グラフ（同時視聴者数＝棒グラフ/左軸、他指標＝線グラフ/右軸）
   - データ駆動の自動考察（ピーク時刻・値・購買転換率）
   - テキストデータからの考察抽出（アドバイスは非表示）

2. **相関の強い組み合わせ**
   - 指標間相関ヒートマップ（正の相関＝緑、負の相関＝赤、同時視聴除く5指標）
   - ピアソン相関係数自動計算
   - 上位3組み合わせカード（相関の方向・スコア・説明文）

3. **各指標の考察とアドバイス_2（複数指標分析）**
   - 分析ポイント②: 商品クリック数の推移（1分単位折れ線グラフ）
   - 分析ポイント③: チャット数の推移（1分単位折れ線グラフ）
   - 分析ポイント④: いいね数の推移（1分単位折れ線グラフ）
   - 各ポイントに考察・アドバイスブロック（データ自動生成 + テキスト分析からの抽出）

#### テキスト処理
- マークダウン記号の自動除去（`*`, `#` など）
- 考察・アドバイスの自動振り分け（キーワードベース）
- 不要文言フィルタリング

### ❌ 未実装機能
- スクリプト文字起こしデータと指標変動の時間軸自動紐付け
- 視聴覚要素（音声・カメラワーク・演出）に基づく深い考察の自動生成
- コメントデータを使った視聴者発言と指標変動の相関分析
- 動画ファイルの実際の分析・活用

## 対応列名（指標データファイル）

| 指標 | 対応する列名（例） |
|------|------------------|
| 経過時間 | 経過時間、elapsed_time、時間（分）、min など |
| 同時視聴ユーザー数 | 同時視聴ユーザー数、viewers、concurrent_viewers など |
| いいね数 | いいね数、likes、reactions など |
| シェア数 | シェア数、shares など |
| チャット数 | チャット数、コメント数、comments、chats など |
| 商品クリック数 | 商品クリック数、product_clicks、item_clicks など |
| カート追加クリック数 | カート追加クリック数、cart_clicks、add_to_cart など |

## 推奨次ステップ

1. **スクリプト文字起こし入力欄の追加**: タイムスタンプ付きの発言データを別入力欄で受け付け、指標ピーク時刻と自動照合
2. **視聴覚要素入力テンプレート**: 音声・視覚・演出の3要素を構造化入力し考察に反映
3. **コメントデータ活用**: コメント頻出ワードと指標変動の相関分析
4. **購買転換フォーカスモード**: 商品クリック→カート追加のファネル分析を強化
