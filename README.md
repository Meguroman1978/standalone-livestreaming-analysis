# ライブ配信エンゲージメント分析ツール

## プロジェクト概要
TikTok Shop / YouTube Live / Instagram Live などのライブ配信データを  
**動画文字起こし（Whisper API）× 指標データ × コメントデータ** で時系列突合し、  
「配信何分時点でどんな発言をしたら指標が急増したか」を具体的に明らかにする  
静的 Web アプリケーションです。

---

## 主要機能

### アップロード
| ファイル | 形式 | 必須 |
|---|---|---|
| 指標データ（時系列エンゲージメント指標） | CSV / XLSX / XLS | ✅ 必須 |
| コメントデータ | CSV / XLSX / XLS | 任意（チェックでスキップ可） |
| 動画ファイル（自動文字起こし） | MP4 / MOV / AVI / m4a / wav | 任意 |

---

### レポート構成

#### 🗺️ 次回配信シナリオ改善サマリー（レポート冒頭）
- **前半/後半トレンドスコアバー** — 各指標が前半集中型か後半集中型かをビジュアル表示
- **改善シナリオ TOP3** — 最も急増幅の大きかった上位3指標について：
  - 急増した時刻・変化率
  - **実際の発言引用（Whisper文字起こし連携時）**
  - 「なぜ急増したか」の考察
  - 「次回シナリオへの組み込み方」の具体的アドバイス

#### 📈 各指標の考察とアドバイス_1（単一指標分析）
- **時系列グラフ** — 同時視聴者数（棒）+ 各エンゲージメント指標（折れ線）
- **考察** — ピーク時間帯・転換率の自動要約
- **全指標アドバイス** — 全指標の急変動ポイントを時系列順に列挙：
  - 文字起こし連携時は **「○分時点の発言 → 考察 → 次回配信アドバイス」** を発言引用付きで出力

#### 🔗 相関の強い組み合わせ（相関分析）
- ピアソン相関係数ヒートマップ（5指標間）
- 相関 TOP3 カード（正/負の相関を色分け）

#### 📊 各指標の考察とアドバイス_2（複数指標分析）
- **分析ポイント② 商品クリック数の推移** — チャート＋考察＋発言引用付きアドバイス
- **分析ポイント③ チャット数の推移** — チャート＋考察＋発言引用付きアドバイス
- **分析ポイント④ いいね数の推移** — チャート＋考察＋発言引用付きアドバイス

#### 📄 PowerPoint (.pptx) ダウンロード
スライド構成（チャート画像＋発言引用付きアドバイステキスト）：
1. タイトルスライド
2. 配信サマリー（数値カード）
3. **次回配信シナリオ改善サマリー TOP3**（発言引用・考察・アドバイス）
4. セクション仕切り（単一指標分析）
5. 分析ポイント① 時系列グラフ
6. セクション仕切り（複数指標分析）
7. 分析ポイント② 商品クリック数（チャート＋発言引用アドバイス）
8. 分析ポイント③ チャット数（チャート＋発言引用アドバイス）
9. 分析ポイント④ いいね数（チャート＋発言引用アドバイス）
10. 相関 TOP3
11. 文字起こし抜粋（取得済み時のみ）

---

## 動画×指標突合分析（Whisper API）
- OpenAI API キーを設定して動画アップロード → Whisper-1 で自動文字起こし（タイムスタンプ付き）
- 各指標が前分比 **30% 以上急増** したポイントを検出
- 急増前後 **±90〜30 秒** の発言を自動抽出
- 考察・次回配信アドバイスをアドバイス欄に直接出力（全レポートセクション共通）
- 文字起こし全文はページ下部のビューワーで確認可能

## データファイルの取得方法

### 指標データファイル（必須）
- **TikTok Shop**: Creator Center → ライブ → 該当配信 → 詳細データ → データをエクスポート (.xlsx)
- **YouTube Live**: YouTube Studio → アナリティクス → リアルタイム → CSV エクスポート
- **Instagram Live**: プロフィール → インサイト → ライブ → CSV ダウンロード

必要な列: `経過時間（分）` / `同時視聴者数` / `いいね数` / `シェア数` / `チャット数` / `商品クリック数` / `カート追加数`

### コメントデータファイル（任意）
- **TikTok Shop**: Creator Center → ライブ詳細 → コメントをエクスポート
- **YouTube Live**: YouTube Studio → コンテンツ → ライブアーカイブ → チャットをダウンロード

---

## ファイル形式の揺れへの対応
30種類以上の列名エイリアスを定義（例: `elapsed_time`, `time(min)`, `concurrent_viewers` など）。  
マッピング失敗時はポジショナルマッピング（1列目=経過時間、2列目=同時視聴）で継続。

## セキュリティ
- アップロードされた全データはブラウザ内のみで処理されます
- 動画ファイルは Whisper API 送信時のみ OpenAI へ送信されます（APIキー設定時）
- APIキーはブラウザのセッション内のみ保持、外部サーバーへは送信されません
