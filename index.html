ライブ配信エンゲージメント分析ツール

動画の自動文字起こし × 指標データ突合 × AI考察で、配信改善を具体的にサポート

OpenAI API キー設定

動画ファイルの自動文字起こし（Whisper API）に使用します。キーはブラウザ上のみで使われ、外部サーバーへは送信されません。
APIキーをお持ちでない場合は platform.openai.com で取得できます。

① 指標データファイル（必須）

  1. TikTok Shop ライブの場合
    TikTok Creator Center → ライブ → 該当配信 → 「詳細データ」→「データをエクスポート」(.xlsx)
  2. Instagram Live の場合
    プロフィール → インサイト → ライブ → 該当配信 → データをCSVでダウンロード
  3. YouTube Live の場合
    YouTube Studio → アナリティクス → リアルタイム → CSVをエクスポート
  4. その他ツール
    StreamLabs・OBS・StreamElements 等から CSV 出力が可能です。
必要な列：経過時間（分）、同時視聴者数、いいね数、シェア数、チャット数、商品クリック数、カート追加数

② コメントデータファイル（任意）

  1. TikTok Shop：Creator Center → ライブ詳細 → 「コメントをエクスポート」
  2. YouTube Live：YouTube Studio → コンテンツ → ライブ配信アーカイブ → チャット再生 → 「チャットをダウンロード」
  3. Instagram：現時点でコメント一括エクスポート機能なし（サードパーティ ツール利用）
コメントデータがない場合は「コメントデータをアップロードしない」にチェックを入れてください。

③ 動画ファイル（Whisper 文字起こし）

  1. 配信アーカイブ動画（MP4/MOV）をそのままアップロード
  2. OpenAI APIキーを設定すると自動で文字起こしを実行
  3. 文字起こし完了後、発言と指標変動を時系列で突合して考察を生成
上限：25MB まで直接送信。超える場合は音声を自動抽出して送信します。

④ 時刻合わせのポイント

  • 指標データの「経過時間」は配信開始からの経過分数で揃えてください
  • 動画も配信開始と同タイミングで録画が始まっていると突合精度が高まります
  • 時刻がずれる場合はテキストエリアに手動で「経過分数」ベースのタイムスタンプを入力してください（例：15:30 この色、肌に馴染む！）
⚠️ 個人情報・機密情報の取り扱い：アップロードされたデータはすべてブラウザ内で処理されます。指標データ・コメントデータ・動画ファイルは外部サーバーへ送信されません（Whisper API 送信時のみ音声データが OpenAI へ送信されます）。
