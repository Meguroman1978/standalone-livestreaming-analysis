# サンプルデータファイル

このディレクトリには、ライブコマース分析ツールをテストするためのサンプルデータが含まれています。

## 📁 ファイル一覧

### 1. streaming_data.csv（配信データ）
配信の時系列指標データ（11分間のサンプル）
- 列: minute, viewers, likes, comments, clicks
- 行数: 11行（0-10分）

### 2. comments_data.csv（コメントデータ）
視聴者コメントデータ（33件のサンプル）
- 列: minute, user, comment
- 行数: 33行

## 🎥 動画ファイルについて

サンプル動画は容量が大きいため、このリポジトリには含まれていません。

**テスト用の動画を準備する方法:**
1. 任意のライブコマース動画を用意（YouTube等からダウンロード可能）
2. MP4形式に変換
3. 10-15分程度の長さを推奨

または、以下のような簡単なテスト動画を作成：
- スマートフォンで商品を撮影
- 10分程度の長さ
- MP4形式で保存

## 🚀 使用方法

1. このサンプルデータをダウンロード
2. 動画ファイルを準備
3. ライブコマース分析ツールにアクセス
4. 3つのファイルをアップロード:
   - 動画ファイル（自分で用意）
   - streaming_data.csv
   - comments_data.csv
5. 分析を実行

## 📊 データの説明

### streaming_data.csv の指標
- **minute**: 経過時間（分）
- **viewers**: 同時視聴者数（100-220の範囲で変動）
- **likes**: いいね数（累積）
- **comments**: コメント数（各時点での数）
- **clicks**: 商品クリック数（各時点での数）

### comments_data.csv の内容
コメントは以下の6カテゴリに分類されるように設計:
- **質問**: 「いくらですか?」「どうやって使う?」
- **驚き**: 「すごい!」「マジで!」
- **ワクワク・期待**: 「欲しい」「気になる」「楽しみ」
- **挨拶**: 「こんにちは」「また来ます」
- **購入意志**: 「買います」「ポチりました」「購入しました」
- **その他**: 上記以外

## 💡 カスタマイズ

これらのサンプルデータは自由に編集・カスタマイズできます:
- 行を追加して配信時間を延長
- 指標の値を変更してピークを作成
- コメントを追加・変更

## 📝 実際のデータを使用する場合

実際のライブコマース配信データを使用する場合:
1. 配信プラットフォームから指標データをエクスポート
2. コメントデータをエクスポート
3. 上記のサンプルと同じ列構造に整形
4. ツールにアップロード

## 🔍 期待される分析結果

このサンプルデータを使用すると、以下のような分析結果が得られます:
- 視聴者数のピーク: 4分目（220人）
- いいね数の増加: 全体的に右肩上がり
- コメントの分類: 質問、購入意志、ワクワクが多い
- クリック数の傾向: 中盤から後半にかけて増加

## ⚠️ 注意事項

- このデータは教育・テスト目的のサンプルです
- 実際の配信データではありません
- 商用利用する場合は実データを使用してください
